# 🟢 Fuzzle Browser 🟢

Fuzzle Browser is a lightweight, futuristic web browser interface built in HTML, CSS, and JavaScript. It features a fully animated, forest-themed UI with a custom new tab page (NTP), quick links, and multi-tab support.  

This project uses [UV Proxy](https://github.com/operasoftware/uv) to safely proxy and render websites inside the browser without cross-origin issues.

---

## Features

- Animated forest-themed gradients and UI elements.
- Multi-tab support with close and navigation controls.
- New Tab Page with a starry background and quick links.
- URL bar with auto-normalization for direct search or site entry.
- Embedded websites run via the UV Proxy to bypass cross-origin limitations.
- Fully responsive design for desktops and tablets.
